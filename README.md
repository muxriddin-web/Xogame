# ❌⭕ Tic Tac Toe Pro

**Tic Tac Toe Pro** is a modern, web-based implementation of the classic Tic Tac Toe game. Designed with a clean, responsive interface and powered by efficient game logic, it delivers a fun and engaging experience for players of all skill levels.

Whether you want to play against a friend on the same device or challenge a smart Bot opponent, **Tic Tac Toe Pro** offers smooth gameplay, modern visuals, and an enjoyable gaming experience.

---

# ✨ Features

## 🤖 Intelligent Bot

Challenge an intelligent Bot opponent designed to provide a fun and competitive experience.

## 📊 Live Statistics

Monitor your gameplay with real-time statistics, including:

- Wins
- Losses
- Draws
- Total Matches
- Match Duration

## 🏆 Scoring System

Track your performance using a built-in scoring system that rewards every victory.

## 🎨 Modern UI/UX

Enjoy a clean and responsive interface featuring:

- Smooth animations
- Responsive layout
- Modern design
- Mobile-friendly experience
- Clean visual effects

## ⚡ Fast Performance

Optimized JavaScript logic ensures fast gameplay and smooth performance across desktop and mobile browsers.

# 🎮 Game Controls

| Action | Description |
|--------|-------------|
| Left Click | Select an empty cell |
| Player Turn | Place **X** or **O** |
| Win | Match ends automatically |
| Draw | Detected automatically |
| Restart | Start a new game instantly |
